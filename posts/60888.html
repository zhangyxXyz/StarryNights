# 引言

为 Shoka 主题添加文章实效性提示 —— 文章发表时间过久，部分内容实际上可能以及过时，需要在显眼处给出浏览者实效性提示。

# 主题集成

需要修改的文件列表如下，相对于 Shoka 主题根目录

# 修改 _config.yml

添加相关控制变量

# 修改多语言字段信息

添加实效性对应文本串，引入格式串文本，多语言适配

# source/js/_app/global.js 添加全局处理函数

# source/js/_app/pjax.js 的 siteRefresh 函数的最后一行 调用全局函数处理

# scripts/generaters/script.js 的 siteConfig 添加变量 (随便选一行后面插入就行，案例写在了 valine 的上面)

# layout/_partials/layout.njk 的 LOCAL 变量添加实效性相关变量

# Front-matter 控制

默认文章实效性检查是启用的，对于不想启用文章实效性的文章

可以在文章头部的 Front-matter 信息强制指定关闭实效性检测

# 效果预览

时效性提示
这是一篇发布于 x 天前，最后一次更新距今已经 y 天的文章，部分内容可能已经过时，请注意甄别。

或者您看到本文时，本文正文顶部已经出现了实效性提示。
